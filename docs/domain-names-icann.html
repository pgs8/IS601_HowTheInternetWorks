Domain names and ICANN

 

A domain name is an online address that offers a user-friendly way to access a website's true online locator: its Internet Protocol (IP) address.

The Internet Corporation for Assigned Names and Numbers (ICANN) is a non-profit organization responsible for managing and coordinating the Domain Name System (DNS), as well as accrediting the domain name registrars.

By ICANN - The logo may be obtained from ICANN., Copyrighted free use, Link
 
 

Why do we use them?

ICANN's management and coordination of DNS ensures that every IP address is unique and that all users of the Internet can find all valid IP addresses. IP Addresses are hard to remember - they're just a string of numbers.

Domain names on the other hand are mnemonic devices that easily identify a company, group, person, etc. Domain names are registered through any one of a number of ICANN accredited domain name registrars.

Entering a domain name into your web browser causes your computer to reach out to its DNS and ask for the IP Address of the domain name you entered. It will then reach out to that server and load the website.
