W3C and HTML and CSS

Oh my!
 

The World Wide Web Consortium (W3C) develops protocols and guidelines that ensure long-term growth for the Web. W3C's standards define key parts of what makes the World Wide Web work.

Hypertext Markup Language (HTML) is the standard markup language for creating Web pages.

Cascading Style Sheets (CSS) describe how HTML elements are to be displayed on screen, paper, or in other media.

 
 

Why do we use them?

W3C standards define an Open Web Platform for application development to enable developers to build rich interactive experiences available on any device. Among the many standards developed are HTML and CSS.

HTML describes the structure of a Web page through a series of elements which tell the browser how to display the content.

CSS saves a lot of work. It can control the layout of multiple web pages all at once.
